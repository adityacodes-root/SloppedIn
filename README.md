# SloppedIn

If the thought exists, there is absolutely no reason it should remain a private thought.

SloppedIn is a Chrome extension that turns whatever you type into an aggressively self-important LinkedIn post.
## What It Does

- Adds a floating `SloppedIn` button on LinkedIn
- Opens a lightweight composer inside the page
- Sends your draft to Gemini
- Returns a polished, narcissistic, deeply unserious LinkedIn post

## What It Is For

This is for:

- inspirational sludge
- lessons learned from objectively deranged situations

This is not for:

- restraint
- touching grass

## Example Input

`i sent the wrong email to the wrong person and somehow made it everyone’s problem`

## Example Output

`I’m excited to share that I sent the wrong email to the wrong person and somehow made it everyone’s problem.`

`Not because it was ideal. Not because it was strategic. But because moments like this reveal who can lead through avoidable chaos.`

`Here are 5 things operators can learn from this experience:`

You get the idea.

## Install

1. Clone this repo
2. Open `chrome://extensions`
3. Turn on Developer mode
4. Click `Load unpacked`
5. Select this folder

## Use

1. Open LinkedIn
2. Click the floating `SloppedIn` button
3. Type something regrettable
4. Click `Slop It`
5. Enjoy the slop
